AZTracker

Программное обеспечение AZTracker предназначено для автоматизации управления задачами в ходе командной работы при выполнении проектов в виде веб-приложения.

AZTracker реализует следующие основные функции:

  • Ведение учета проектов
  • Ведение учета пользователей и групп
  • Ведение учета поставленных задач
  • Отображение сводных сведений о ходе выполнения проектов
